# Mobile App

Flutter shell for the Nexthria live-painting workflow.

## Current Phase

Phase 0 wires the capture surface, operator guidance overlay, scratch-card mosaic preview, utility and coverage meters, and export summary card.

## Run

```bash
git clone https://github.com/dharmiclink/retina-mosaic.git
cd retina-mosaic
dart pub get
dart run melos bootstrap
cd apps/mobile
flutter run
```
