# IP And Freedom-To-Operate Constraints

This codebase is constrained to the Nexthria method of action:

- continuous extraction from a video stream
- real-time frame utility filtering
- non-sequential dynamic projection onto a live reference canvas
- operator guidance without discrete capture events

Explicitly excluded from the default workflow:

- auto-trigger shutter logic
- stop-and-shoot capture orchestration
- fixed burst montage sequencing
